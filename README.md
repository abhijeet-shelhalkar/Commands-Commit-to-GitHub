#### Commands: for New repository

```js
- git init
- git add .
- git commit -m "msg"
- git branch -M main
- git remote add origin <git url>
- git pull origin main --allow-unrelated-histories
- git push -u origin main
```

#### Commands: for existing repository

```js
- git remote add origin <git url>
- git branch -M main
- git pull origin main --allow-unrelated-histories
- git push -u origin main
```
